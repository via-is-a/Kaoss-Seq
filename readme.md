# Kaoss-Seq

Kaoss-Seq is a random MIDI sequencer. This supports 31 different scale patterns like a Korg Kaossilator.

# DEMO

"hoge"の魅力が直感的に伝えわるデモ動画や図解を載せる

# Features

* Random sequence
* 31 scales(Kaossilator style)
* Key setting
* 12 pattern memory
* BPM shuffle
* MIDI output

# Requirement


* Purr-data(NOT vanila)
